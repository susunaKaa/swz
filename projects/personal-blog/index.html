个人博客系统

基于 Jekyll 的现代化个人博客

Jekyll HTML5 CSS3 JavaScript GitHub Pages
2024年01月

项目简介

这是一个使用 Jekyll 静态网站生成器构建的个人博客系统。网站采用现代化的设计理念，具有以下特点：

  • 响应式设计：完美适配各种设备尺寸
  • 优雅的排版：注重阅读体验
  • 快速加载：静态网站，加载速度极快
  • SEO 友好：优化的元数据和结构化数据
  • 易于维护：使用 Markdown 撰写文章

技术栈

  • 前端框架：原生 HTML5, CSS3, JavaScript
  • 构建工具：Jekyll 4.3
  • 样式预处理：Sass/SCSS
  • 部署平台：GitHub Pages
  • 版本控制：Git

主要功能

  1. 首页展示：展示最新文章和个人介绍
  2. 博客列表：按时间倒序展示所有文章
  3. 项目展示：展示个人项目作品
  4. 关于页面：个人简介和技能展示
  5. 联系表单：方便访客联系

开发过程

设计阶段

在设计阶段，我注重用户体验和视觉美感的平衡。选择了渐变色作为主色调，营造现代感。

开发阶段

使用 Jekyll 的模板系统，创建了可复用的布局和组件。通过 Sass 实现了模块化的样式管理。

部署阶段

利用 GitHub Pages 进行托管，实现了自动化部署。每次推送代码，网站会自动更新。

未来计划

  • 添加深色模式支持
  • 集成评论系统
  • 添加搜索功能
  • 优化 SEO
  • 添加更多交互动画

总结

这个项目让我深入了解了静态网站生成器的工作原理，以及如何构建一个高性能、易维护的个人博客系统。
